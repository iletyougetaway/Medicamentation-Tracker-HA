# Быстрый старт

Менеджер лекарств настраивается из интерфейса Home Assistant. YAML для
лекарств, NFC-меток, напоминаний и истории не используется.

## Установка

1. Установите интеграцию через HACS как пользовательский репозиторий.
2. Перезапустите Home Assistant.
3. Откройте **Настройки** > **Устройства и службы**.
4. Добавьте интеграцию **Менеджер лекарств**.
5. В параметрах интеграции укажите сервис уведомлений `mobile_app`, если нужны
   автоматические напоминания.

## Карточка Lovelace

После установки через HACS интеграция сама добавляет ресурс карточки:
`/medication_manager/frontend/medication-manager-card.js?v=0.7.1`.

Если панели управления работают в YAML mode, добавьте этот JavaScript module в
ресурсы вручную.

Откройте нужную панель управления и добавьте карточку **Менеджер лекарств** из UI.

Карточка показывает список лекарств, статус на сегодня, следующее напоминание,
последний приём и недельную историю. Лекарства можно добавлять и редактировать
из самой карточки.

## NFC-метки

NFC-метка привязывается к лекарству в диалоге добавления или редактирования.
После сканирования интеграция сама обработает событие Home Assistant `tag`:
автоматизации для этого не нужны.

## Напоминания

Планировщик интеграции сам создаёт callback для каждого включённого напоминания
и отправляет уведомление через выбранный сервис `mobile_app`. Действия в
уведомлении позволяют отметить приём или отложить напоминание.

## Недельная история

В недельной истории принятый приём отображается иконкой лекарства, поздний
приём отмечается жёлтым индикатором, пропущенный день остаётся пустым, а будущие
дни показываются как неактивные.
